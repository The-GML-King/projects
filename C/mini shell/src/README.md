# MiniShell

![C](https://img.shields.io/badge/language-C-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![Status](https://img.shields.io/badge/status-learning-yellow)

Enjoy this shell that I made with the sole purpose of learning C language.

> It's only compatible with windows, nothing big or significant.

## Features
- Has a functional help page.
- Can run explorer using syscalls.
- Can run notepad using syscalls.
- Can run the explorer's Run dialog (a.k.a win + r).
- Can access time and date.
- It's essentially light so don't worry about your hard drive.
